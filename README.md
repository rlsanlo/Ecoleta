# Ecoleta
 Projeto que visa, cadastrar e identificar pontos de coleta de resíduos.
---

# Linguagem e Frameworks
---
* HTML
* CSS
* JavaScript
* NodeJS
* Express
* Template engine ( Nunjucks )
* Banco de dados ( SQLLite3 )
